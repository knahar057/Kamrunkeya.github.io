Kamrun Nahar Keya website package for GitHub Pages.

Upload these files to a GitHub Pages repository:
- index.html
- style.css
- folders: research, teaching, publications, programs, seminars, vitae
- CV file as cv.pdf
- headshot as profile.jpg

After upload:
1. Make sure GitHub Pages is set to deploy from main / root.
2. Refresh the site after 1-2 minutes.
3. To show the photo on the homepage, edit index.html and replace the placeholder div with:
   <img src="profile.jpg" alt="Kamrun Nahar Keya" class="photo" />
